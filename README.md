# 📊 Portfolio Tracker — Gestão de Investimentos & Dividendos

Uma ferramenta gratuita e open-source para gestão de portfolios de investimento, focada em dividendos, análise de ações/ETFs e planeamento financeiro.

**🔗 [Abrir App](https://k1mmis.github.io/portfolio-tracker/)**

## 🔒 Privacidade & Segurança

- **Zero servidores** — A app corre 100% no teu browser, nada é enviado para lado nenhum
- **Sem login, sem registo** — Não existe sistema de contas nem autenticação
- **Dados no localStorage** — Ficam exclusivamente no TEU computador/browser
- **Sem cookies de tracking** — Zero telemetria, zero analytics
- **Open source** — Todo o código é visível e auditável
- **Cada utilizador isolado** — Se partilhares o link, a outra pessoa vê dados de exemplo, nunca os teus

## ✨ Funcionalidades

- 📰 Feed de notícias (RSS) e vídeos YouTube com prioridade e filtro 7 dias
- 📊 Dashboard com gráficos de alocação, P&L, dividendos, meta de dividendos
- 📈 Gestão de ações com auto-fetch por ticker, conversão EUR/USD automática
- 🏦 ETFs organizados por planos editáveis com links JustETF
- 💰 Calendário de dividendos com análise de meses fortes/fracos
- 🔬 Analisador de ações (preços, 52w high/low, gráfico 1M-5A)
- 🔬 Analisador de ETFs (retornos, volatilidade, Sharpe Ratio)
- 📐 Matriz de correlação entre ações do portfolio
- 🔥 Calculadora FIRE/Coast FIRE com cenários
- 📄 Relatório mensal em PDF
- 📊 Importação CSV de XTB e Trading 212
- 👀 Watchlist separada para ações e ETFs
- 🔗 Links úteis organizados por categoria
- 💾 Sistema multi-conta, export/import JSON, validação de dados

## 🚀 Como Usar

1. Abre [https://k1mmis.github.io/portfolio-tracker/](https://k1mmis.github.io/portfolio-tracker/)
2. A app vem com dados de exemplo para explorar
3. Edita ações, ETFs e planos conforme o teu portfolio
4. Importa posições via **Importar CSV** (suporta XTB e Trading 212)
5. Usa **Exportar JSON** regularmente para backup (a app avisa após 7 dias sem backup)

## 🛠️ Stack Técnica

HTML + CSS + JS puro (ficheiro único) | Chart.js | jsPDF | Yahoo Finance API (via CORS proxy com fallback) | RSS2JSON | GitHub Pages
